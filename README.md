# library-app
### Описание проекта

Веб-приложение «Библиотека» - это учебный проект, разработанный на платформе ASP.NET Core MVC. Основная цель приложения - демонстрация реализации CRUD-операций (создание, чтение, обновление, удаление) с использованием паттерна MVC и Entity Framework Core.

### Что делает приложение

**1. Просмотр списка книг**
На главной странице отображается таблица со всеми книгами, которые добавлены в систему. Пользователь может увидеть название, автора, год издания и жанр каждой книги.

**2. Добавление новой книги**
Через форму добавления пользователь может внести новую книгу в список. Для названия и автора установлена обязательная проверка — если оставить эти поля пустыми, приложение покажет сообщение с просьбой заполнить их. Остальные поля (год издания и жанр) заполняются по желанию.

**3. Редактирование существующей книги**
Рядом с каждой книгой в списке есть кнопка редактирования. При нажатии открывается форма, в которой уже подставлены текущие данные книги. Пользователь может изменить любые поля и сохранить обновлённую версию.

**4. Удаление книги**
Кнопка удаления позволяет убрать книгу из списка. Удаление происходит сразу без дополнительных подтверждений, что делает процесс быстрым и удобным.

### Как это устроено внутри

**Модель Book**
Это класс, который описывает структуру данных. В нём есть поля: идентификатор (Id), название (Title), автор (Author), год издания (Year) и жанр (Genre). Для Title и Author добавлены атрибуты обязательности, чтобы форма не пропускала пустые значения.

**Контроллер BooksController**
Здесь сосредоточена вся логика работы с книгами. Контроллер содержит методы для:
- вывода списка всех книг,
- показа формы добавления,
- обработки отправленной формы добавления,
- показа формы редактирования,
- обработки изменений,
- удаления записи.

Все методы, которые работают с данными, сделаны асинхронными, чтобы приложение оставалось отзывчивым даже при большом количестве записей.

**Представления (Razor Pages)**
Интерфейс реализован с помощью Razor-синтаксиса. Есть отдельные страницы для списка книг, для добавления и для редактирования. Формы используют встроенную валидацию, которая подсказывает пользователю, если что-то заполнено неверно.
